How It Works

Rose uses these specialized agents to handle your requests:

Master Orchestrator Agent

Central coordinator that processes all incoming user requests and routes them to the appropriate specialized agent. It can sequence multi-step operations and manage the flow of information between agents.

Mail Agent

Handles email interactions including retrieving, filtering, and summarizing emails. Also converts emails into structured commands for Asana project/task creation.

Slack Agent

Manages all Slack operations including channel creation, user management, message retrieval and sending, conversation summarization, and channel archiving.

Asana Agent

Handles all Asana project and task management including creation, modification, listing, and deletion of projects and tasks.

Calendar Agent

Manages Google Calendar operations including creating, updating, searching, and retrieving events and meetings.

Quick Start Commands

Input Methods

  • Slack: Type commands directly in any channel where Rose is added
  • Telegram: Send text or voice messages to the Rose chatbot
  • Email: Send emails that trigger automated workflows

Slack Commands

  • Create channel #project-name - Creates a new Slack channel
  • Get messages from #channel-name - Retrieves recent messages
  • Send message to #channel-name: your message - Posts a message

Asana Commands

  • Create project called Project Name - Creates a new Asana project
  • Add task Title with description Details to Project Name - Creates a task
  • List all tasks in Project Name - Shows all tasks in a project

Email Commands

  • Get emails from sender@example.com - Retrieves emails from sender
  • Summarize emails about "Topic" - Summarizes relevant emails

Calendar Commands

  • Schedule meeting "Title" on June 30 at 2pm - Creates calendar event
  • Show my meetings for tomorrow - Lists upcoming events

Example Workflows

Convert Email to Asana Tasks

Command: Convert email from sender@example.com to Asana tasks in Project Name

This automatically creates Asana tasks from email content with proper titles and descriptions.

Slack to Asana Workflow

Command: Get messages from #project-planning, summarize them and create an Asana project with tasks

This captures Slack discussions and converts them into actionable Asana tasks.

Calendar Management

Command: Schedule meeting with team about project launch on Friday at 2pm for 1 hour

This creates a calendar event with all specified details.

Pro Tips

  • Use action verbs: Start commands with create, get, schedule, send, or list
  • Reference channels with #: Always use # before Slack channel names
  • Be specific with names: Use exact project, task, and event names
  • For complex tasks: Use "then" to chain commands ("Get emails then create tasks")
  • Voice commands: Speak naturally to Telegram for audio responses
